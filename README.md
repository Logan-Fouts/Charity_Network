# Charity_Network
Prototype Charity Network Application

## Requirments:
* textrazor  
    * ```pip install textrazor```
* mysql.connector
    * ```pip install mysql-connector-python```
* secrets
    * secrets.py
        * ```MYSQL_HOST```
        * ```MYSQL_USER```
        * ```MYSQL_PASSWORD```
        * ```MYSQL_DATABASE```
        * ```API_KEY```
